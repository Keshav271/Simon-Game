Game Play Instructions


  • Firstly, the game shows the first colour in the sequence (let:blue). The user clicks on the blue button.

  • Next, the game shows the next colour (let:red), the user has to remember the sequence is blue, red and has to click in same sequence (starting from blue).

  • The game level up and the next colour blinks(let:yellow), the user has to click the sequence blue, red, yellow and so on and so forth

  • If the user messes up the sequence, then the game ends, and by pressing any key can restart.
